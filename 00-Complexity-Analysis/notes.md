# My Notes - Complexity Analysis

## O(1)

Input kitna bhi ho, operations same rehte hain.

Example:
- print(x)

---

## O(n)

Input badhega to operations bhi utne badhenge.

Example:
- Single for loop

---

## O(n²)

Nested loops.

Outer × Inner

---

## O(log n)

Har iteration me data half ya double hota hai.

Example:
- Binary Search

---

## O(n log n)

Mostly efficient sorting algorithms.
log n times n elements pr algorithm lga rhe hai.


---

## O(2ⁿ)

Har step par branching.

Example:
- Recursive Fibonacci
- Backtracking

---

## Space Complexity

Interview me generally Auxiliary Space count karte hain.

Extra memory only.

---

## In-place Algorithm

Original data ko hi modify karta hai.

Extra memory:
O(1)

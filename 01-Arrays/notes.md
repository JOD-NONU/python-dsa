# Arrays

## What is an Array?

Array is a linear data that stores multiple elements in contigous memory locations.

## Memory Representation

Arrays store elements in contiguous memory locations, meaning all elements are placed one after another in memory.
Each element occupies a fixed amount of memory depending on its data type.

### Example
Array:
[10, 20, 30, 40, 50]


## Address Formula

The address of any element can be calculated using the formula:
Address = Base Address + (Index × Size of Data Type)

## Static vs Dynamic Array

Static Array has fixed size whereas Dynamic Array size can change automtically.
Static Array are faster than Dynamic Array
Static array are C/C++ arrays and Dynamic Array are Python List

# Hashing

## What is Hashing?

Hashing is a technique used to store and retrieve data efficiently using key-value pairs.

Average Operations:

- Search → O(1)
- Insert → O(1)
- Delete → O(1)

---

## When to use Hashing?

✔ Frequency Counting

✔ Duplicate Detection

✔ Fast Lookup

✔ Character Mapping

✔ Grouping Similar Data

✔ Complement Search
